# Cowork - 機能一覧

凡例: [実装済] / [一部実装] / [未実装] / [将来]

## コア機能

- [実装済] **チャットインターフェース** — Claudeとの日本語対話、ストリーミング表示、ウェルカム画面
- [実装済] **ツール実行の日本語翻訳** — Bash, Read, Write, Edit, Glob, Grep等の操作をリアルタイムで日本語表示
- [一部実装] **承認ダイアログ** — 権限確認を日本語で表示（UIコンポーネントは存在するが、Claude Codeのストリームとの接続が未実装）
- [実装済] **セッション継続** — `--resume` + session_id保存による会話の継続
- [実装済] **ファイルブラウザ** — 作業フォルダ内のファイル一覧・ツリー表示

## ワークフロー自動化（コマンド）

- [実装済] **コマンド管理** — よく使う手順をClaude Code標準コマンド（`.claude/commands/*.md`）として保存・実行・削除
- [実装済] **コンテキスト自動提供** — 作業フォルダ・選択ファイルを自動でコマンドに渡す。`$ARGUMENTS` で受け取り
- [実装済] **ファイル選択** — ファイルブラウザでチェックボックス選択、コマンド実行時にコンテキストとして自動提供
- [実装済] **レガシーマイグレーション** — 旧スキル（JSON形式）から自動変換
- [将来] **コマンドのインポート/エクスポート・共有** — .mdファイルなのでそのままコピー可能
- [未実装] LLMコンポーネントを含むワークフローのコマンド定義

## Slack連携

- [実装済] **OAuth2認証** — 埋め込みClient IDによるOAuth2フロー、ローカルコールバックサーバーで自動コード受信
- [実装済] **Slack Lists API** — リストアイテムの取得・作成（Bot Token認証）
- [実装済] **設定UI** — 「Slackに追加」ボタン1つで認証、リストID設定
- [未実装] **Socket Mode** — リアルタイムイベント受信（パブリックURLが不要なデスクトップアプリ向き）
- ※Reminders APIは2023年3月に非推奨。Lists API（2025年9月リリース）を使用

## Google Drive連携

- [実装済] **OAuth2認証フロー** — 埋め込みClient IDによるOAuth2フロー、ローカルコールバックサーバーで自動コード受信
- [一部実装] **ファイル一覧** — バックエンドAPI実装済み、ファイルブラウザUIからの閲覧は未実装
- [一部実装] **ファイルダウンロード** — バックエンドAPI実装済み、UIからの選択・ダウンロード操作は未実装
- [実装済] **設定UI** — 「Googleでログイン」ボタン1つで認証（Client ID/Secret手動入力は不要）

## TODO管理

- [実装済] **ローカルTODOリスト** — 追加・完了トグル・削除・永続化（バックエンド側）
- [未実装] **Slackリストからの取得・表示・操作** — バックエンドAPI（list_items/create_item）は実装済み、TODO画面からの操作UIは未実装
- [未実装] **ローカルTODOとSlackの双方向同期** — ローカルTODOとSlackリストの同期ロジックは未実装

## ファイルの整理

- [未実装] ルールに基づくファイル自動仕分け（例：請求書を年月フォルダに振り分け）
- [実装済] フォルダ構成の把握と説明（チャット経由で可能）
- [将来] **ルールエンジン** — ファイル仕分けルールのGUI定義

## 外部サービス連携

- [未実装] **ドキュメント管理SaaS連携**（Docubase等） — API経由でのドキュメント取得・操作
- [未実装] **契約管理SaaS連携** — APIがないサービスの場合、スキルとして手順を定義し半自動化
- [未実装] **メール処理** — メール内容の整理、返信文案の作成
